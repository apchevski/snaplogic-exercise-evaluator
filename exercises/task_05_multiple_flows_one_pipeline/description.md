# Task 05 – Multiple Flows – One pipeline

### Objective:

There are 3 reports to be created based on the file “Trng_Employees.csv” file provided.

**Report 1**: The first report needs to contain only the records that have an even index.

- File Name: “Trng_FilterEven.csv”.

**Report 2**: Once you have the condition to filter only the records with an even index. Create a file which will contain only records where the value in the ID field is less than 15.

- File name: “Trng_IDbelow15.csv”.

**Report 3**: Same objective as Report 2 but this time create a file containing the records where the value in the ID field is bigger than 15.

- File name: “Trng_IDabove15.csv”.

### Note:

In Objective, by [even] index we mean index as a record in a database. Every record has its own unique id (index). Essentially you need to add an index to each record, find a way to do it. There are two ways to achieve this.
