# Task 03 – Join Employee Records

### Objective:

Based on the two data sources containing sales leads and income information, generate a report associating each sale lead with the income distribution of their zip code. The report needs to contain records only for leads from San Francisco. The end goal of this requirement is for us to determine the income information for each zip code where each lead lives.

Additionally, please provide the data in ascending order based on their Zip code.

Target file name: Task3.csv.

### Task Instructions:

File Names for this task:

“Leads.csv” - list of individuals and their contact information including their zip code.

“CAIncomeByZip.csv” - provides income information by the California zip code.
