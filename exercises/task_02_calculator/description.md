# Task 02 – Calculator

### Objective:

Create a pipeline that will calculate for simple math equations.

### Task Instructions:

- Add a pipeline parameter named mathOperation and a value of “3+5”.
- Get the operator from the above parameter and calculate the equation and return the the result in the format below:
- [
- {
- "result": "3 + 5 = 8"
- }
- ]
- If there is no operator provided return the following response: “No operator in the equation”.
[

{

"result": "No operator in the equation"

}

]

- Create Triggered task and test it for different equations.
