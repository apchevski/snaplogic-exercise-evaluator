# Task 04 – Born on Friday

### Objective:

We need to get random users from a public API. Work with that list so we can get the desired output.

### Task Instructions:

1. Retrieves data for 100 random users from the API: https://randomuser.me/api/
    a. Note: For every call to the API you get one random user.

2. Filter users that are 40 years old and older, and that are born on a Friday.

3. As a final output, your pipeline should return the following fields:
    a. Full Name
    b. City
    c. Country
    d. DOB – formatted to: dd-MM-yyyy
    e. Age
    f. Day Of Week - Always Friday

4. Output the data into an excel file called Born_on_Friday.xlsx.
