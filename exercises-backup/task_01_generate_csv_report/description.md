# Task 01 – Generate CSV Report

### Objective:

Based on the zip file (Task1.zip) provided, please create a CSV report (file) named “CA_Birthdays.csv”. The report needs to contain the columns last name, first name, and birthdays. Additionally, the records need to be alphabetized by last name in descending order, but the report needs to be only for residents of California (The city is not relevant in this case).

### Task Instructions:

Do not unzip the file manually – find a snap that will unzip the file for you and let the pipeline do it.

### Bonus Question:

You will work with over 4,000 records, but the preview will contain far fewer. Why?
